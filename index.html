ReadMate App

Description

ReadMate is an e-book reading app that lets users browse, purchase, and read e-books from Firebase and external APIs. Features an admin module for managing book collections, FCM notifications, and full Firebase integration.

Key Features

  • Library (Firebase books) & Explore (API books) categories.
  • Purchase and read e-books within the app.
  • Bookcase feature to save books for later.
  • User reviews for library books.
  • Download books as PDFs from the Explore section.
  • Payment tracking & transaction history.
  • Personalized push notifications via Firebase Cloud Messaging.
  • Authentication via Google or email/password.

Technical Stack

Kotlin MVVM & StateFlow Firebase Auth Firestore Firebase Cloud Messaging Firebase Storage Retrofit & OkHttp Koin DI DBooks API ViewPager2
